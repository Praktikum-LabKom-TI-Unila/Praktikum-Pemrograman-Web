## Cara Menggunakan

Untuk file HTML (Modul 1, 3, 5, 6):
1. Buka file HTML langsung di browser.

Untuk file PHP (Modul 4 dan 6.3):
1. Jalankan server bawaan PHP dari folder root:
   php -S localhost:8000
   Akses: http://localhost:8000/4/4.1/hello.php
2. Atau jalankan dari folder modul:
   cd 4/4.1
   php -S localhost:8000
   Akses: http://localhost:8000/hello.php
3. Tekan Ctrl + C untuk menghentikan server.

Catatan:
- Ganti port jika bentrok (php -S localhost:3000).
- Server ini hanya untuk pengembangan lokal.

## Catatan Tambahan
